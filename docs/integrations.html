Integrations

How Nobulex plugs into the agent stack.

MCP (Model Context Protocol)

Add trust to any MCP server. Nobulex middleware intercepts tool calls and enforces covenants before they execute. Quickstart shows the basic pattern.

ElizaOS

Nobulex plugin available for ElizaOS. Add accountability to Eliza agents with minimal config.

LangGraph / OpenAI tools

Wrap your agent’s tool layer with Nobulex. The SDK’s protect() pattern works with any tool-calling framework. See GitHub for examples.
